# 运营报告分析技能 (Operations Report Analysis Skill)

## 背景 (Background)

房地产企业运营管理需要定期（如每周、每两周）召开敏捷运营会议，跟踪各项目和公司整体的经营指标完成情况、项目进展、风险卡点等。这些会议报告通常以PowerPoint (`.pptx`) 格式呈现，包含：

- 财务指标完成情况（销售额、回款、现金流、工抵等）
- 各项目群/项目维度的KPI达成情况
- 重点项目进展与卡点
- 组织管理优化举措
- 存量资产盘活进展
- 跨项目协调事项

**问题**：PPTX格式的报告难以进行跨期对比分析、数据提取和趋势分析。人工阅读多份报告并总结分析效率低下。

## 目的 (Purpose)

本技能旨在自动化处理和分析企业运营报告，实现：

1. **格式转换**：将PPTX格式报告转换为Markdown格式，便于文本处理和分析
2. **时间序列分析**：处理多期时间序列的报告，识别趋势和变化
3. **公司层面总结**：输出公司整体经营动态的执行摘要（非单个项目视角）
4. **事实与观点分离**：先陈述客观事实，再进行专业点评

## 输入数据 (Input Data)

### 主要输入

1. **PPTX运营报告文件**
   - 文件命名示例：`【财务运营】敏捷会报告1222.pptx`
   - 文件名中包含日期信息（如1222 = 2025年12月22日）
   - 文件内容包含运营会议的完整信息

2. **时间序列信息**
   - 多期连续的报告文件
   - 按时间顺序排列
   - 覆盖指定的分析周期

### 数据要求

- 报告应为同一系列（如同为"【财务运营】敏捷会报告"）
- 时间序列应连续，无明显断层
- 报告结构相对稳定，便于对比分析

## 工作流程 (Workflow)

### 阶段1：PPTX转Markdown（如需要）

当用户提供PPTX格式报告时，Claude Code应执行以下步骤：

1. **定位转换脚本**
   ```bash
   python scripts/pptx_to_md.py <pptx_folder_or_files> -o output-md
   ```

2. **批量转换**
   - 如果用户指定文件夹，转换该文件夹内所有PPTX文件
   - 如果用户指定具体文件，仅转换指定文件
   - 输出MD文件保存到`output-md`子目录

3. **验证转换结果**
   - 确认所有文件已成功转换
   - 检查输出文件数量与输入一致

### 阶段2：读取时间序列报告

1. **识别相关文件**
   - 根据文件名前缀或关键词筛选目标文件（如"【财务运营】"）
   - 按日期顺序排序文件

2. **读取报告内容**
   - 逐份读取MD文件内容
   - 理解每份报告的结构和关键信息

### 阶段3：分析与报告生成

1. **事实陈述 (先陈述事实)**
   - 整体财务经营指标完成情况
   - 重点项目进展与卡点分析
   - 组织管理优化举措
   - 存量资产盘活进展
   - 跨项目协调事项

2. **点评与分析 (再加以点评)**
   - 财务维度的积极表现与问题
   - 项目运营中的深层次问题
   - 组织管理与执行力问题
   - 战略层面的思考

3. **总结与建议**
   - 核心发现（亮点、问题、根源）
   - 改进建议

## 分析要求 (Analysis Requirements)

### 公司层面视角

- **聚焦整体**：分析公司整体经营动态，而非单个项目详情
- **跨项目对比**：分析不同项目群之间的表现差异
- **系统性问题**：识别跨项目、跨职能的共性问题

### 事实与观点分离

- **事实陈述**：
  - 客观呈现数据和事件
  - 引用具体数字和完成率
  - 描述实际发生的进展和延期

- **点评分析**：
  - 基于事实的专业判断
  - 识别趋势和模式
  - 分析根本原因
  - 提出改进方向

### 关键关注维度

| 维度 | 关键指标 |
|------|----------|
| 财务 | 自销完成率、权益回款、自由现金流、工抵完成率 |
| 项目 | 重点项目节点达成、交付风险、卡点问题 |
| 管理 | 组织调整、考核优化、风险管控 |
| 资产 | 存量盘活进展、决策推进情况 |
| 协调 | 融资突破、合作方沟通 |


## 技术说明 (Technical Notes)

### PPTX转换脚本

- **位置**: `scripts/pptx_to_md.py`
- **依赖**: python-pptx
- **功能**:
  - 提取每张幻灯片的标题和内容
  - 识别段落、表格、项目符号
  - 转换为结构化的Markdown格式
  - 支持中文字符（UTF-8编码）

### 依赖安装

```bash
pip install python-pptx
```

## 注意事项 (Important Notes)

1. **日期解析**: 文件名中的日期格式需要正确识别（如1201 = 2025-12-01）
2. **公司视角**: 避免陷入单个项目的细节，聚焦公司整体动态
3. **事实客观**: 事实陈述部分必须客观准确，数据来源可追溯
4. **分析深度**: 点评分析要有深度，不能停留在表面现象
5. **建议可行**: 改进建议应具体可操作，而非泛泛而谈
